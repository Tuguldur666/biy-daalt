## Төслийн бүтэц
```
todo_app/
├── app/
│   ├── __init__.py      # Flask app factory
│   ├── models.py        # User, Task загварууд
│   └── routes.py        # API endpoint-ууд
├── tests/
│   └── test_api.py      # 20 Test Cases (unittest)
├── run_tests.py         # Нарийвчилсан тайлан
├── requirements.txt
└── README.md
```

## Windows дээр суулгах, ажиллуулах

```cmd
:: 1. Хавтас руу орох
cd todo_app

:: 2. Виртуал орчин үүсгэх (заавал биш)
python -m venv venv
venv\Scripts\activate

:: 3. Сангуудыг суулгах
pip install -r requirements.txt

:: 4. Тест ажиллуулах — нарийвчилсан тайлантай
python run_tests.py

:: 5. Эсвэл стандарт unittest-ээр
python -m unittest tests\test_api.py -v
```

## API Endpoint-ууд

| Method | Endpoint | Тайлбар |
|--------|----------|---------|
| POST | /auth/register | Бүртгэл |
| POST | /auth/login | Нэвтрэх |
| GET | /tasks | Даалгавруудын жагсаалт |
| POST | /tasks | Шинэ даалгавар |
| GET | /tasks/<id> | Нэг даалгавар |
| PUT | /tasks/<id> | Засах |
| DELETE | /tasks/<id> | Устгах |
